ByteKing Javascript Client
============
This library provides an API to track events and update profiles on ByteKing.

Install with Composer
------------
The main repository is hosting https://github.com/startbase/byteking-js

Use npm to install:
```bash
npm install byteking-js
```
As a dependency Byteking agent also has to be installed. Read more about agent here:
https://github.com/startbase/byteking-nodejs-agent
